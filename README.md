This project is a static website clone developed to practice and showcase frontend development skills. It replicates the visual identity of the Zoya boutique, known for its diamond jewellery from the House of Tata. The project includes a fully responsive Home page and an About page.

Responsive Navbar: Fixed navigation bar with hover animations and mobile compatibility.
