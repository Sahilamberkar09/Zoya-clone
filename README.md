## ðŸ’Ž About the Project
This project is a static website clone developed to practice and showcase frontend development skills. It replicates the visual identity of the Zoya boutique, known for its diamond jewellery from the House of Tata. The project includes a fully responsive **Home** page and an **About** page.

---

## ðŸ›  Tech Stack

* **HTML5**: Semantic structure and layout.
* **CSS3**:
    * Flexbox & CSS Grid for layouts.
    * CSS Variables (`:root`) for color consistency.
    * Media Queries for responsiveness (Mobile, Tablet, Desktop).
    * Animations and Hover effects.
* **FontAwesome**: For UI icons (Search, Cart, Social Media).
* **Google Fonts**: Typography (AGaramondPro-Regular replication).

## âœ¨ Key Features

* **Responsive Navbar**: Fixed navigation bar with hover animations and mobile compatibility.
* **Dynamic Collections**: Vertical scrolling marquee animations to showcase different jewellery collections.
* **Video Background**: "MySong" section featuring a muted, looping background video with overlay text.
* **Interactive Grids**:
    * "Discover By Categories" section with circular image styling and hover borders.
    * "Spotted in Zoya" celebrity showcase using a mixed grid layout.
* **Newsletter Section**: Styled subscription form with a parallax-effect background.
* **Multi-page Navigation**: Linked `index.html` (Home) and `about.html` (About) pages.
* **Custom Footer**: Detailed footer with quick links, services, and social media integration.
